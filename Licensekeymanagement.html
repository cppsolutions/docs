License key management

Named licenses

Named licenses can only be used by one specific person and installed on her own computer. To validate the correct use of the license, CPP Analyzer needs to be activated on the computer where it will be used. To activate CPP Analyzer, simply enter the license key you've retrieved from your email.


Warning: Activating CPP Analyzer requires internet access to connect to the license server. Once activated, CPP Analyzer will check the license server from time to time. If it can't connect to the license server, it will keep working for a certain period of time (usually,7 days). Once this period is over, it won't run  and will require access to the license server.

How to: Activate CPP Analyzer

Once you've purchased a named license, you will be able to download the full version of CPP Analyzer,
When you launch your licensed copy of CPP Analyzer, it will automatically ask you to enter your license key and connect to the license servers to validate it. Once this is done, CPP Analyzer will be activated on that computer and you won't need to enter the license key again. It will periodically check with the license server to make sure the key is still valid.
