Viewing the Results DashBoard

To navigate to the Results dashboard either use:

•the Ribbon Tabs > View Tab > Click the "Results dashboard" button. or

•the Configuration panel > Click the "Results dashboard" button.


The Results dashboard is comprised of three different views:

•CPP Results

•CPP-OAS Results

•Survivor benefit results


Each of these views allow the user to visually compare the results from scenarios A and B by viewing the information either graphically (bar graphs and graphs) or in table form.


Each of the three views displays:

1.Two bar graphs, one for each scenario showing the annual benefit per year up until the clients longevity age.

2.One graph plotting the accumulated benefit for each year allows you to see the break even age (if there is one) for the two scenarios.

3.Two tables showing both the annual and accumulated benefit amounts, one table displays the discounted values and the other the present day values.


Note: The Survivor benefit results view will display three possible results based on the client's situation:

•No results if the client is not eligible for the survivor benefit.

•Survivor benefit  (Break even) - if the client's spouse has passed, useful for understanding how the uncombined survivor benefit is combined with the CPP benefit.

•Survivor benefit (Estate planning) - If the client's spouse is still alive, useful for future planning.





1. The left panel of each view displays quick results for the accumulated values at the client's longevity age and break even ages.

2. The Variables section at the bottom allows you to change CPP inflation, Discount rate, and the client's longevity age by moving the sliders.

▪These changes are changed for the whole client profile.

3. The View Table | Graphs at the button of the view allows you to switch in between the Graphs and Table views.

4. Control flow:


•The  button at the bottom right moves you to the next view.

•The  button at the bottom move you to the previous view (Not all views).

•The  button exits the dashboard. (ONLY IN SURVIVOR BENEFIT VIEW)
