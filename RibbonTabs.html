Ribbon Tabs

CPP Analyzer's ribbon tabs are located at the top of the main window and provide all the features available within CPP Analyzer in a categorized fashion.

Each TAB will display actions grouped by similar purpose.

The Top ribbon is made of four tabs:

oFile

oHome

oView

oHelp

Home tab


The "Home" tab provides access to the 7 steps of client configuration:

•Step 1 - Client details - Rates

•Step 2 - CPP disability

•Step 3 - Child rearing

•Step 4 - Statement of contributions

•Step 5 - Client scenarios

•Step 6 - Survivor benefit

•Step 7 - Client report


View tab



The "View" tab gives access to:

•View Dashboard.

•View PRB analysis

•View Detailed Calculation for scenario A and B

•View archived client reports

Help tab



The "Help" tab provides access to :

•Printable User's Guide

•Online help

•About CPP Analyzer and Contact information
